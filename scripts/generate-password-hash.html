Parola Hash Üretici

Damla ve Hilal için kullanacağımız parolaları PBKDF2-SHA256 yöntemiyle güvenli bir hash değerine dönüştürür.

Bu dosyayı yalnızca hash üretmek için kullan. İşlem bittikten sonra GitHub reposundan sil. Parolanın kendisini hiçbir SQL dosyasına veya JavaScript dosyasına yazma.
